
# 🚀 AI Document Analyzer

### 📄 Turn PDFs into Insights, Answers & Visualizations

> ✨ Upload documents. Extract intelligence. Visualize insights.

---

## 🧠 Overview

**InsightFlow AI** is a GenAI-powered document intelligence platform that transforms unstructured PDFs into meaningful insights using **Retrieval-Augmented Generation (RAG)**.

It allows users to:

* 📄 Upload PDFs
* ❓ Ask questions
* 📊 Get structured answers + visual analytics

---

## 🔥 Features

* 📂 **PDF Upload & Processing**
  Extracts and processes text from documents in real-time

* 🧠 **RAG Pipeline**
  Combines embeddings + vector search for accurate answers

* 🔍 **Semantic Search (FAISS)**
  Finds the most relevant context intelligently

* 🤖 **LLM Integration (Groq / LLaMA)**
  Generates clean, structured responses

* 📊 **Graph & Insights Visualization**
  Displays similarity scores and analytics

* ⚡ **Fast & Interactive UI**
  Built with React + Tailwind

---

## ⚙️ How It Works

```text
PDF → Text Extraction → Chunking → Embeddings → FAISS
      ↓
User Query → Semantic Search → Relevant Context
      ↓
LLM → Structured Answer + Insights + Graphs
```

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/insightflow-ai.git
cd insightflow-ai
```

---

### 2️⃣ Backend Setup

```bash
cd backend
python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt
uvicorn main:app --reload
```

---

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
